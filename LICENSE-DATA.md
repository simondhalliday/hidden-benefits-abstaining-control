# Data License

The data and documentation in this draft replication package are intended to be released under the Creative Commons Attribution 4.0 International License (CC-BY 4.0), subject to final coauthor review and data-rights confirmation.

You are free to share and adapt the released data with attribution once the package is finalized. Until the Zenodo release is created, treat this repository as a public working draft rather than the archival version of record.

Raw zTree session files, payment records, reimbursement records, administrative files, literature PDFs, and old paper drafts are not part of this public data release.
