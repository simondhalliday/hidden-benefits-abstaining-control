# Coauthor Review Packet

This draft replication package is for:

Gabriel Burdin, Simon D. Halliday, and Fabio Landini, "The hidden benefits of abstaining from control," *Journal of Economic Behavior & Organization*, 147, 1-12, 2018.

Repository: https://github.com/simondhalliday/hidden-benefits-abstaining-control

## Proposed public contents

- Consolidated analysis data only.
- Stata code needed to reproduce the published analysis.
- Final zTree program files.
- Final instructions, questionnaire, and appendix materials.
- Source archive manifest documenting excluded and review-pending files.

## Not proposed for public GitHub

- Raw zTree session outputs.
- Session questionnaire spreadsheets.
- Payment, receipt, reimbursement, and administrative records.
- zTree executables.
- Literature PDFs.
- Old paper drafts and referee-response files.

## Decisions requested

1. Can the consolidated analysis data be released publicly under CC-BY 4.0?
2. Should raw zTree files be deposited on Zenodo, deposited with restricted access, or excluded from the public release?
3. Are comparison data from Falk-Kosfeld, Ploner-Schmelz-Ziegelmeyer, or Schnedler-Vadovic redistributable in this package?
4. Are there any IRB, consent, or lab-policy restrictions that should be noted in the README?
5. Should the package be archived on Zenodo with all three coauthors listed?
