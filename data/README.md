# Data

This draft includes consolidated analysis files only:

- `processed/merged_treatment_control.dta`: Stata dataset used by the legacy master script.
- `processed/merged_control_treatment.csv`: CSV version of the Burdin-Halliday-Landini experimental data identified in the project archive.
- `processed/merged_kf_ziegel.csv`: Pooled comparison data used in the legacy master script.

The raw zTree session files, questionnaire spreadsheets, payment files, and pilot files are not included in this public draft. They remain in the private source archive pending privacy and coauthor review.

See `../DATA_PROVENANCE.md` and `../docs/audit/source_manifest.csv` for details.
